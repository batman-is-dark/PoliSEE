# poliSEE

Policy simulation frontend and backend.

Frontend: `frontend/` (Vite + React + TypeScript)
Backend: `backend/` (Python)

To run frontend locally:

1. cd frontend
2. npm install
3. npm run dev

To initialize a GitHub repo and push:

1. git init
2. git add .
3. git commit -m "Initial commit"
4. Create a repository on GitHub and add the remote, then `git push -u origin main`
